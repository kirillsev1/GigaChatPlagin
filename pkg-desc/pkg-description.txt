Это плагин, который расширяет функционал LibreOffice. 
Он умеет менять стилистику написанного текста, определять главные идеи текста, исправлять в нем ошибки. Для использования достаточно выделить текст и выбрать нужную команду.
Для своей работы плагин использует нейронную сеть GigaChat от Сбербанка.

Для работы вам потребуется токен для работы с API GigaChat
https://developers.sber.ru/portal/products/gigachat

Приятного пользования.
